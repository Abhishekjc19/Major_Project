📱 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

Authentication:
  POST   /api/auth/register       Create account
  POST   /api/auth/login          Get JWT token
  GET    /api/auth/me             Current user (protected)

Buses:
  GET    /api/buses               List all buses (with filters)
  GET    /api/buses/:id           Get bus details
  POST   /api/buses/:id/occupancy Update occupancy (conductor)
  POST   /api/buses/:id/location  Update location (conductor)
  POST   /api/buses/:busId/stops/:stopId/mark-passed (conductor)

Health:
  GET    /api/health              Server status

Socket.IO Events:
  buses:request-update            Get bus data
  ticket:sold                     Add passenger
  ticket:cancelled                Remove passenger
  bus:location-update             Update GPS
  bus:stop-passed                 Mark stop reached
  chat:message                    Send message
  sos:activate                    Emergency alert
  sos:update-status               Update SOS (conductor)
